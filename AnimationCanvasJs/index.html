Algorithms

Selection Sort

Selection sort is an in-place comparison sort. It has O(n2) time complexity, making it inefficient on large lists, and generally performs worse than the similar insertion sort.

Binary Search Tree

A binary search tree is a rooted binary tree, whose internal nodes each store a key and each have two distinguished sub-trees, commonly denoted left and right. It is a special type of binary tree in which left child of a node has value less than the parent and right child has value greater than parent.
